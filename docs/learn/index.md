# Learning Path

Welcome to the Dream Learning Path. This series of lessons will take you from "Hello World" to building production-ready applications with advanced patterns.

**Time:** Designed to be completed in a short session  
**Prerequisites:** [Gleam installed](https://gleam.run/getting-started/installing/) and basic familiarity with the language.

---

## The Path

### 1. [Hello World](01-hello-world.md)
Build your first server. Understand the core pieces: Router, Controller, and Server.
- **Key Concepts:** `router`, `route`, `bind`, `listen`

### 2. [Building an API](02-building-api.md)
Create a REST API with JSON responses. Learn how to handle data and errors.
- **Key Concepts:** `json_response`, `Services`, `Result` error handling

### 3. [Adding Auth](03-adding-auth.md)
Secure your API. Learn how to use middleware and custom context.
- **Key Concepts:** `Middleware`, `Context`, `AuthContext`, `get_header`

### 4. [Advanced Patterns](04-advanced-patterns.md)
Scale your application logic. Learn the Operations pattern for complex business logic.
- **Key Concepts:** `Operations`, Service Coordination, Transaction Management

---

## What's Next?

Once you've completed the path, you'll have a solid understanding of Dream's core philosophy. From there, you can:

- **Build something specific:** Check out the [Guides](../guides/) for task-based help.
- **See real code:** Explore the [Examples](https://github.com/TrustBound/dream/tree/main/examples/) to see working applications.
- **Deep dive:** Read the [Reference](../reference/) docs for API details.
